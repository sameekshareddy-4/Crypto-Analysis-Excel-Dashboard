# Crypto Analysis Excel Dashboard

## Project Overview

This project is an Excel-based Cryptocurrency Analysis Dashboard developed as part of a Data Analyst Internship. It analyzes cryptocurrency market data using Pivot Tables, Pivot Charts, Slicers, KPIs, XLOOKUP, and VBA to generate interactive dashboards and business insights.

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- XLOOKUP
- VBA (Macros)
- Conditional Formatting

---

## Dataset

The dataset contains approximately 200 cryptocurrency records including:

- Coin Name
- Symbol
- Current Price
- Previous Price
- 1 Hour Change
- 24 Hour Change
- 7 Day Change
- Market Capitalization
- Trading Volume
- Circulating Supply

---

## Tasks Completed

### Task 1 – Time-Based Visibility Dashboard
- Top 10 cryptocurrencies by trading volume
- Dynamic chart visibility using VBA
- Dashboard updates automatically

### Task 2 – Crypto Coin Comparison
- Compare any two cryptocurrencies
- Displays price, market cap, volume and circulating supply
- Dynamic comparison using XLOOKUP

### Task 3 – Liquidity Analysis
- Top 5 cryptocurrencies by trading volume
- Interactive Pie Chart
- Price range filtering using slicers

### Task 4 – Price Change Analysis
- Top 10 cryptocurrencies ranked by price change
- Interactive chart
- Price range slicer
- Coin information displayed using KPIs

### Task 5 – Historical Price Change Insights
- Comparison of 7-Day and 24-Hour price changes
- Top 10 coins ranked by 1-Hour change
- Interactive slicer for price range

### Task 6 – Low Budget Investment Insights
- KPI Dashboard
- Displays:
  - Coin Name
  - Symbol
  - Current Price
  - Average Downfall
  - Total Coins
- Interactive price range slicer
